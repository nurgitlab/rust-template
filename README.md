# To get started
This is Hello World temlapte for Rust and VSCode.

## Language install

```bash
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Project init

```bash
$ cargo new hello_cargo
$ cd hello_cargo
```
